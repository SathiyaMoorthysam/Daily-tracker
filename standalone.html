ℹ️ Setup Instructions

  1. Open Google Sheets and create a new blank sheet.
  2. Go to Extensions → Apps Script.
  3. Replace the default code with the contents of Code.gs.
  4. Click Deploy → New deployment → Web app.
  5. Set Execute as: Me. Set Who has access: Anyone.
  6. Copy the Web App URL and paste it above.
  7. Click Test Connection to verify.
